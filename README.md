# Student Grade Analyzer

A Python-based CLI tool that analyzes student grades from a CSV file.

## Features
- Calculate total marks, average, and letter grade for each student
- Identifies pass/fail status
- Shows class statistics (mean, max, min, std deviation) usinf NumPy
- Displays top 3 students and failed students
- Exports full report to CSV

## Tech Stack
- Python
- NumPy
- Pandas

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the analyzer:
   python analyzer.py

## OUTPUT
- Printed report in terminal
- Exported report saved to data/report.csv
